Privacy Policy for WhisperWall

Effective Date: August 6, 2025

Developer: Arfan

1. Information We Collect

  • Anonymous Messages: Users can post text anonymously. No personally identifiable information (PII) is collected.
  • Reactions: Emoji reactions to messages are stored using unique anonymous IDs.
  • Firebase Analytics (Optional): May collect aggregated usage data to improve the app.

2. Use of Information

We use the data to:

  • Display and organize anonymous posts.
  • Enable emoji reactions.
  • Improve app functionality.

3. Data Security

We use Firebase Firestore, which encrypts data both at rest and in transit. User anonymity is preserved with randomly generated IDs.

4. Third-Party Services

WhisperWall uses:

  • Google Firebase (Firestore & optional Analytics)
  • Shared Preferences (for storing device settings locally)

These third parties have their own privacy policies.

5. Children's Privacy

WhisperWall is not intended for users under 13. No age verification is performed, but the app collects no personal data.

6. Changes to This Policy

We may update this policy from time to time. Changes will be posted on this page.

7. Contact

For questions or concerns, contact the developer via the GitHub repository.
